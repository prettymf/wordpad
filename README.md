# Wordpad

# 

# Простой текстовый редактор, реализованный на Delphi с использованием TRichEdit.

# 

# Возможности

# Открытие и сохранение файлов (.txt, .rtf)

# Копирование / вырезание / вставка

# Выбор шрифта (ComboBox и FontDialog)

# Выравнивание текста (влево, по центру, вправо)

# Базовое форматирование текста

# Технологии

# Delphi (Object Pascal)

# VCL (TRichEdit, диалоги, кнопки)

# Запуск

# 

# Открыть файл проекта .dpr в Delphi и запустить (F9).

# 

# Примечание

# 

# Файлы .rtf сохраняют форматирование, .txt — обычный текст.
